#WeatherApp
A real-time weather dashboard built using Java Spring Boot and HTML/CSS/JavaScript.

##Features
-Search weather by city
-Real-time temperature
-Humidity and wind details
-Responsive UI

##Technologies Used
-Java
-SprinBoot
-HTML
-CSS
-JavaScript
